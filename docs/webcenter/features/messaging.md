# Messaging

> **Stub.** Internal messaging between firm members.

## Composing a message

1. Go to **Messages → Compose**.
2. Select recipients — individuals, an office, a role, or all employees.
3. Write the message. The editor supports basic rich text (bold, italics, lists, links).
4. Send.

## Inbox and outbox

- **Inbox** — Messages you've received. Unread messages appear bold.
- **Outbox** — Messages you've sent.

## Message of the Day

Administrators can set a Message of the Day under **Settings → MOTD**. It shows on the home page until dismissed by each employee.

## Privacy

Messages stay within your firm. WebCenter does not surface messages from other firms, even for employees who have linked accounts at multiple firms.
